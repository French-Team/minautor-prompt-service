## Problème transverse : tous les services sont en RAM

```
TemplateLibraryService  → private templates = new Map<string, PromptTemplate>()
VersionHandler          → private versionStore = new Map<string, PromptVersion[]>()
PromptManager           → private promptStore = new Map<string, GeneratedPrompt>()
IdentityResolver        → private currentIdentity: UserIdentity | null = null
```

**Aucune persistance nulle part.** Un redémarrage du serveur = tout est perdu.
