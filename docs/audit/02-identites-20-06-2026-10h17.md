| Persistance | ❌ **Aucune** | Rien en localStorage, base de données ou API |

## Problèmes

- **Aucune persistance** : refresh de la page → tout est perdu
- **Aucun backend** : pas de connexion à un service d'authentification (Auth0, Keycloak, etc.)
- **Données 100% en mémoire** : les profils, capacités et permissions sont des tableaux JavaScript statiques
- **Pas de gestion utilisateur réelle** : pas de login, pas de session, pas de rôles dynamiques

## Notes pour dev-plan

- Choix du backend : Auth0, Keycloak, Firebase Auth, ou solution custom ?
- Les identités devraient être persistées (localStorage au minimum, backend au mieux)
- Les profils User/Superviseur/Responsable sont un bon point de départ conceptuel
